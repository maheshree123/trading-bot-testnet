# Binance Futures Trading Bot (Testnet)

This is a simplified Python-based trading bot built using the Binance Futures Testnet API.

## Features
- Market and Limit orders
- Buy and Sell support
- Command-line interface
- Logging of API requests, responses, and errors
- Error handling for failed API calls

## Tech Stack
- Python
- python-binance
- Binance Futures Testnet (USDT-M)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the bot:
   python bot.py

3. Enter your Binance Testnet API key and secret when prompted.

## Notes
- This bot works only on Binance Futures Testnet.
- Do NOT use real trading credentials.
